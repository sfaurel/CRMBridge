Documentación de la API

Bienvenido

La documentación de la API está disponible en el endpoint /api.

Instrucciones

  • POST /users: Crear un usuario
  • GET /leads: Acceder a los leads (requiere inicio de sesión)
  • POST /auth/login: Iniciar sesión
  • GET /auth/logout: Cerrar sesión
  • GET /status: Consultar el estado de la API

Se necesita un token JWT para interactuar con los endpoints protegidos. Utiliza el endpoint /auth/login con las credenciales de usuario y copia el token (Authorization) devuelto en el botón "Authorize" de la documentación.

Para obtener más detalles sobre el uso de la API consulta la documentación.
